Примеры переиспользуемых компонентов

Документы

Компонент работает с массивом документов или с массивом групп.

Вход:

@Input isViewMode - режим просмотра без добавления, удаления документов
@Input documentsToShow - массив документов для отображения
@Input documentsToShowByGroup - массив объектов групп:
            group_name: string;
            documents: Document[];
Выход:
@Output detachDocument - возвращает объект открепленного документа
@Output attachDocument - возвращает объект прикрепленного документа
@Output currentDocumentList - возвращает массив объектов документов


AttachDocumentDialog
При вызове диалога можно передать конфиг. В поле data добавляем объект с необязательными полями:
selectedDocuments - массив документов, которые будут выделены. если мод single, то один документ
attachMode: 'single'  - если хотим получить диалог с радиобаттонами
